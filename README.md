# 📊 Analyse des Salaires — Offres Data & IA France

## 🎯 Objectif
Pipeline ETL complet sur Databricks pour analyser les salaires
des métiers Data & IA en France.

## 🛠️ Technologies
- PySpark
- Databricks
- Delta Lake
- SQL

## 📋 Étapes
1. Génération de 1500+ offres d'emploi
2. Nettoyage avec PySpark
3. Stockage Delta Lake
4. Analyse SQL (salaires par métier, ville, entreprise)

## 📈 Résultats
- 1500+ offres analysées
- Salaire moyen par métier et par ville
- Top entreprises qui recrutent
